# Treat the Disease, Not the Symptom Examples

Use this folder for sanitized examples of company diagnostic memos.

Good examples should include:

- Presenting symptom
- Likely disease
- Diagnostic evidence
- Pattern match
- Hidden work
- Second-order consequences
- Severity rating
- Tests to run
- Blunt internal and meeting-ready questions
- Treatment plan
- Watchouts

Initial examples to add later:

- Rebrand under-scope
- Second product drag
- Strategy theater
