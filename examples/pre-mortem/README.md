# Pre-Mortem Examples

Use this folder for sanitized examples of pre-mortem memos.

Good examples should include:

- Plan being tested
- Imagined failure
- Most likely failure modes
- Named pattern matches
- Weak assumptions
- Mitigations
- Leader asks
- Watchpoints
- Revised confidence

Initial examples to add later:

- Launching a second product next quarter
- Moving a customer-facing workflow to a new system
